# Walmart Weekly Sales Forecasting

## Overview
Time Series forecasting project to predict weekly sales for Walmart stores.

## Technologies Used
- Python
- Pandas, NumPy, Matplotlib, Seaborn
- Scikit-learn, Statsmodels (ARIMA)

## Project Structure
- `Walmart_Weekly_Sales_Forecasting.ipynb` → Main notebook
- `walmart_train.csv`, `features.csv`, `stores.csv` → Datasets
- `requirements.txt` → Dependencies

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Open `Walmart_Weekly_Sales_Forecasting.ipynb`
3. Run all cells
